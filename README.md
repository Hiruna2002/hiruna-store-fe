-- Hiruna Store --

Hiruna Store is a modern online clothing shop built using the MERN Stack.
It allows users to browse products, view details, and purchase clothes online with a smooth user experience.

-- Tech Stack --

This project was built using the MERN Stack:

* React (Frontend)

* Node.js

* Express.js

* MongoDB

* Tailwind CSS
* Redux

* JWT Authentication

  Features

-- Browse clothing products --

* View product details

* User registration & login

* Add to cart functionality

* Order management

* Secure authentication

* Responsive design
